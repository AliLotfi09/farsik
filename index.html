چگونه کار می‌کند؟

این سیستم از الگوریتم پیشرفته‌ای برای تشخیص کلمات انگلیسی استفاده می‌کند. ابتدا کلمات رایج و الگوهای خاص (مانند لینک‌ها و ایمیل‌ها) را بررسی می‌کند، سپس با استفاده از API دیکشنری صحت کلمات را تأیید می‌کند.

کلمات فارسی که با کیبورد انگلیسی تایپ شده‌اند (مانند "sghl" که باید "سلام" شود) تشخیص داده شده و به فارسی تبدیل می‌شوند.
